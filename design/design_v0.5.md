# 项目管理系统增量设计 v0.5

## 新增功能：每周视图事项状态过滤

### 功能概述
修改每周视图，添加按状态过滤事项，帮助用户更好地聚焦不同阶段的任务。

### 使用场景
- **聚焦未完成**: 只查看待办和进行中的事项，专注当前工作
- **回顾已完成**: 查看本周已完成的事项，总结工作成果
- **关注特定状态**: 快速定位某个状态的所有事项
- **组合筛选**: 结合项目和状态筛选，精准查找

---

## UI 设计

### 每周视图布局调整

```
[每周视图]
│ 筛选条件:                                                    │
│ [项目: 全部项目 ▼]  [状态: 全部状态 ▼]  [清除筛选]        │
├─────────────────────────────────────────────────────────────┤
│ 本周事项列表 (共 12 项)                                      │
│                                                              │
│ [事项 1]                                                 │
│ [事项 2]                                                 │
│ [事项 3]                                                 │
│ ...                                                          │
└─────────────────────────────────────────────────────────────┘
```

### 状态筛选器设计

#### 1. 筛选栏布局
```
┌──────────────────────────────────────────────────────┐
│ 筛选:                                                 │
│ [📁 项目: 全部项目 ▼]  [🏷️ 状态: 全部状态 ▼]        │
│                                       [🔄 清除筛选]    │
└──────────────────────────────────────────────────────┘
```

#### 2. 状态下拉菜单选项
```
🏷️ 状态: 全部状态 ▼
├─ 全部状态 (显示所有)
├─ 📋 待办
├─ 🔄 进行中
├─ ✅ 已完成
└─ ⏸️ backlog
```

#### 3. 筛选状态显示
当应用筛选时，在筛选栏下方显示当前筛选状态：
```
┌──────────────────────────────────────────────────────┐
│ 当前筛选: 项目「前端开发」+ 状态「进行中」            │
│ 共找到 5 个事项                              [清除 ×] │
└──────────────────────────────────────────────────────┘
```

### 视觉反馈

#### 1. 筛选激活状态
- 未应用筛选：下拉框使用默认灰色边框
- 已应用筛选：下拉框使用蓝色边框，显示选中的筛选条件

#### 2. 空状态提示
当筛选后无结果时显示：
```
┌──────────────────────────────────────────────────────┐
│                    🔍                                 │
│              未找到符合条件的事项                      │
│                                                       │
│         尝试调整筛选条件或清除筛选重新查看             │
│                                                       │
│                  [清除筛选]                           │
└──────────────────────────────────────────────────────┘
```
